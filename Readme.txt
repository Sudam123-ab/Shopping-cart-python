* Shopping Cart - Python CLI App


* Features
- View products with stock management
- Add/Remove items from cart
- Coupon system (SAVE10, SAVE20, FLAT500)
- Auto stock update
- Checkout with final bill
- Order history saved to CSV
- Input validation and error handling

* Technologies Used
- Python 3
- File Handling (CSV)
- Exception Handling
- DateTime module
